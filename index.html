01
Concept

感覚に、データという根拠を。Science the talent

野球は、感覚だけでは強くなれない。PUMPSは、まるでスポーツ版の人間ドックのように一人ひとりの身体能力と動作を計測。 強みと伸びしろを「見える化」し、技術・思考・身体を科学的に鍛えます。

  • 01
    技術を磨く動作解析・データ計測でフォームを最適化
  • 02
    思考を鍛える野球IQ・判断力で「考えて動ける」を育てる
  • 03
    身体をつくるフィジカル強化と障害予防を両立

02
Measurement

5つの基礎能力を、測る。5 fundamental abilities

PUMPS式スポーツドックでは、選手の土台となる5つの能力を計測。レーダーチャートで強みと課題が一目でわかります。

A-01

打力 / POWER

スイング速度と打球初速。ミート力と飛距離の源泉を計測します。

A-02

球速 / VELOCITY

球速・回転数・リリースを数値化し、投球の質を可視化します。

A-03

走力 / SPEED

スプリントタイムと一歩目の加速。試合を動かす脚を測ります。

A-04

敏捷性 / AGILITY

反応・方向転換の速さ。守備と走塁を支える俊敏性を評価します。

A-05

体幹 / CORE

下半身と体幹の安定。すべての動作を支える土台の強さを測定します。

REPORT

個別レポート

5項目を1枚にまとめ、次のトレーニングへ繋げます。

03
Program

3つの指導領域Hit · Throw · Run

計測データをもとに、一人ひとりに最適化したメニューを設計。「打つ・投げる・走る」を個別に伸ばします。

01 / 打つ

バッティング

スイングを計測し、データで分解。ミート力・飛距離・再現性を、一人ひとりのフォームに合わせて引き上げます。

  • スイング解析
  • 打球速度計測
  • 個別フォーム設計
02 / 投げる

ピッチング

球速・回転数・フォームを科学する。肩肘に優しい投げ方を身につけながら、球速アップとコントロールを両立します。

  • フォーム解析
  • 球速・回転数
  • 障害予防
03 / 走る

走塁・フィジカル

一歩目の速さと判断力は、練習でつくれる。スプリント技術・走塁判断・体幹強化で、試合を動かす脚をつくります。

  • スプリント
  • 走塁判断
  • 体幹・下半身
04
Flow

うまくなる仕組みMeasure → Analyze → Evaluate → Train

PUMPS式・育成サイクル。測って終わりではなく、分析・評価し、トレーニングへ繋げ、再計測で成長を確認します。

01
Measure

測定

スイング・球速・走力・敏捷性・体幹を計測しデータ化。

02
Analyze

03
Evaluate

評価

04
Train

実践

少人数指導で反復。再計測で成長を確認し次へ。
