CONCEPT

広島から、
世界基準の育成を。

野球は、感覚だけでは強くなれない。 PUMPSは一人ひとりの動きを計測・分析し、技術も思考も身体も鍛える。 広島のグラウンドから、世界で通用する選手を送り出します。

  • 01 技術を磨く — 動作解析・データ計測
  • 02 思考を鍛える — 野球IQ・判断力
  • 03 身体をつくる — フィジカル・障害予防

PROGRAM

3つの指導領域

01 / 打つ

バッティング

スイングを計測し、データで分解。ミート力・飛距離・再現性を一人ひとりのフォームに合わせて引き上げます。

  • スイング解析
  • 打球速度計測
  • 個別フォーム設計
02 / 投げる

ピッチング

球速・回転数・フォームを科学する。肩肘に優しい投げ方を身につけながら、球速アップとコントロールを両立します。

  • フォーム解析
  • 球速・回転数
  • 障害予防
03 / 走る

走塁・フィジカル

一歩目の速さと判断力は、練習でつくれる。スプリント技術・走塁判断・体幹強化で、試合を動かす脚をつくります。

  • スプリント
  • 走塁判断
  • 体幹・下半身

METHOD

うまくなる仕組みを、回す。

スクロールでボールが回転し、PUMPS式・育成サイクルが一周します。

  1. 01
    体験 / TRIAL無料体験で、今の実力と課題を知る。
  2. 02
    計測 / MEASUREスイング・球速・走力をデータ化。
  3. 03
    分析 / ANALYZE強みと伸びしろを見える化。
  4. 04
    設計 / PLAN一人ひとりの個別メニューを設計。
  5. 05
    実践 / TRAIN少人数指導で反復し、技術を習得。
  6. 06
    検証 / REVIEW再計測で成長を確認し、次の目標へ。
